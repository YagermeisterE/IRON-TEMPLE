# Роль

Ты Senior Full Stack Developer, UI/UX Designer и Frontend Architect с большим опытом разработки современных веб-приложений.

Твоя задача — разработать полноценный учебный проект для курсовой работы студента колледжа по модулю **ПМ.05 (Frontend-разработка)**.

Не создавай примитивный лендинг. Сайт должен выглядеть как реальный коммерческий проект 2026 года.

---

# Тематика

Разработать современный сайт фитнес-клуба.

Название клуба:

**IRON TEMPLE**

---

# Цель проекта

Создать адаптивный современный сайт с использованием:

* HTML5
* CSS3
* JavaScript (ES6+)

Разрешается использовать:

* LocalStorage
* Fetch API
* Font Awesome
* Google Fonts
* AOS Animate
* Swiper
* Chart.js

Не использовать Bootstrap.

Весь дизайн должен быть написан самостоятельно.

---

# Требования к дизайну

Стиль:

Премиальный современный фитнес-клуб.

Цветовая схема:

* черный
* графит
* темно-серый
* белый
* красный (#E53935)

Использовать:

* Glassmorphism
* Плавные анимации
* Hover эффекты
* Красивые переходы
* Blur эффекты
* Скругленные карточки
* Современные кнопки
* Адаптивность

Визуально сайт должен выглядеть на уровне современных сайтов Nike, Gymshark или популярных фитнес-клубов.

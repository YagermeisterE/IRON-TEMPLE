# Project Summary
The project "IRON TEMPLE" is a comprehensive multi-page fitness club website designed to provide users with an engaging platform for fitness services. It features a premium dark design with glassmorphism effects, and is built solely with HTML5, CSS3, and JavaScript (ES6+), ensuring a seamless user experience without any backend or frameworks. The site includes essential functionalities such as an admin panel, local storage for data persistence, and is fully responsive and SEO optimized.

# Project Module Description
The project consists of several functional modules:
- **Core Layout**: Manages the header, footer, and navigation.
- **Styling**: Implements the dark theme, glassmorphism effects, and responsive design.
- **Forms**: Handles signup form validation and submission.
- **Storage**: Manages LocalStorage CRUD operations for form submissions.
- **Gallery**: Displays images in a grid format with lightbox functionality.
- **Slider**: Features a testimonials carousel.
- **Admin Panel**: Allows management of form submissions with search and sorting capabilities.

# Technology Stack
- **HTML5**: For semantic structure and markup.
- **CSS3**: For styling and layout (including glassmorphism).
- **JavaScript (ES6+)**: For interactivity and functionality.
- **Libraries**: 
  - Font Awesome: Icons
  - Google Fonts: Typography
  - AOS (Animate on Scroll): Scroll animations
  - Swiper: For sliders
  - Chart.js: For displaying statistics in the admin panel

# Usage
To set up the project, follow these steps:
1. **Install Dependencies**: Navigate to the project directory and run:
   ```
   pnpm install
   ```
2. **Build the Project**: Run the build command to create the production files:
   ```
   pnpm run build
   ```
3. **Run the Development Server**: Start the server to view the project locally.
